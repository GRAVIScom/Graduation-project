Добрый день, дорогой пользователь!

Функционал данного сайта позволяет переводить числа, символы и строки во внутренний формат типов данных языков программирования С# и Pascal

Руководство пользователя

Чтобы перевести число , символ или строку:

  • выберите режим *обучения*
  • введите число(вещественные числа вводите через запятую)
  • выберите тип данных
  • нажмите кнопку *перевод*

Для проверки своих знаний:

  • выберите режим *контроль*
  • выберите тип
  • нажмите кнопку *сгенерировать*
  • переведите число, символ или строку
  • введите в поле для ввода
  • нажмите кнопку *проверить*
